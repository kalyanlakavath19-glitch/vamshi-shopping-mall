# Vamshi Shopping Mall - Responsive Retail Website
A modern and responsive shopping mall website developed using HTML, CSS, and JavaScript.
the website is designed to showcase products, customer reviews, payment methods, location, and contact information for Vamshi Shopping Mall.

## Features
- Responsive design for mobile and desktop
- Attractive hero section
- product collection for men, women, and kids
- customer reviews section
- Google Maps location Integration
- WhatsApp and phone call buttons
- Multiple navigation menu
  ## Technologies Used
  - HTML5
  - CSS3
  - JavaScript
  - Front Awesome for Images
  - Google Maps Embed
